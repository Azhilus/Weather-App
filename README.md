## Getting Started

1. Clone the repository:

   ```bash
   git clone https://github.com/Azhilus/Weather-App.git
   ```

2. Navigate to the project folder:

   ```bash
   cd Weather-app
   ```

3. Install dependencies:

   ```bash
   flutter pub get
   ```

4. Run the app:

   ```bash
   flutter run
   ```
